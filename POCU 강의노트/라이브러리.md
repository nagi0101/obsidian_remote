## 라이브러리
---
- 함수 등을 기계어로 변환 후 파일 하나로 저장해둔 것
- 나중에 다른 .c에서 기능이 필요할 때 링크해서 쓸 수 있음
- 두 종류가 존재
	- 정적 라이브러리
	- 동적 라이브러리

### 정적 라이브러리
- 라이브러리 안에 있는 기계어를 최종 실행파일에 가져다가 복사
- 동적 링킹에 비해
	- 실행 파일 크기 큼
	- 메모리 더 잡아먹을 수 있음
	- 실행 속도 빠름
		- 컴파일러와 링커가 최대한 최적화 진행
